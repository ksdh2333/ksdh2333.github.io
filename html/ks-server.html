Ks-server

介绍

一个开在自己电脑上时不时就会掉线重启的服务端, 还有可能被Ks-game影响。

开服务端原本只为了使用MCDReforged的插件和Velocity互通服的方便, 但是也可以用来联机。
但是通常没有人会想要加入, 一是服务器不稳定, 二是不会为了玩游戏而耗费过多精力。

如果有人加入服务器, 那么这个服务器本质上就是是朋友联机,
所有人为了同一个目标(比如建造机器)忙碌, 但是在空闲时间也可以聊天, 并不会强迫任何人干活, 除非TA违反了规定。

服务器现有 3 个分服, main(主服), mirror(镜像服), crea(创造服)

主服
生存主服, 生电服本体
镜像服
主服的镜像, 不定期同步主服存档, 模式为创造, 用于刷怪塔测试、坐标测试等。请不要将镜像服与主服搞混
创造服
虚空存档, 创造模式, 用于机器、建筑设计测试

生存服死亡掉落

插件/模组

服务端现为 Java 版 1.21fabric , 使用 MCDReforged 代理, 并使用 Velocity 建立服务器联通

  • Velocity插件

    maintenance
    为服务器添加维护模式, 启用维护模式时, 普通玩家不能进入服务器
    minimotd
    为服务器在服务器列表的文字添加彩色等效果
    Reconnect
    将玩家重新连接到上次退出前连接的子服

  • MCDReforged插件

    Beep
    为服务器添加@玩家的功能
    CrashRestart
    服务端崩溃时自动重启
    DayCountNBT
    !!day可以查看服务器开服时间
    MirrorMcsmcdR(仅主服)
    用于自动同步主服存档到镜像服
    PrimeBackup
    全图备份插件, 每天定时备份, !!pb make手动创建备份
    Region-BackUp
    局部备份插件, 主服很少使用, 会回档地形但不回档玩家
    simple_op
    !!op以获取2级服务器op权限(仅镜像服、创造服)

  • fabric模组

    carpet, Carpet-AMS-Addition, carpet-extra, carpet-org-addition, carpet-tis-addition, CrystalCarpetAddition, gugle-carpet-addition
    为创造设计提供便捷性功能, 重新引入更新抑制、旧版袭击机制, 添加假人等功能
    XaerosWorldMap, Xaeros_Minimap
    让客户端XaerosWorldMap、Xaeros_Minimap更兼容, 并能无视距离显示其他玩家位置
    Jade
    让客户端Jade更兼容, 能获取到蜜蜂等服务端信息
    Lithium
    优化服务端Mspt
    noisium
    优化区块加载和生成造成的卡顿
    pca-protocol
    让客户端MasaGadget更兼容, 能获取到村民相关信息等服务端信息
    quickrightclick, Advanced Shulkerboxes
    让玩家可以对空气右键打开潜影盒、工作台以及末影箱
    servux
    让客户端Masa系列模组更兼容, 能获取到结构范围、史莱姆区块等服务端信息
    syncmatica
    用于与客户端syncmatica配合共享投影原理图

规则

客户端功能

仅服主允许的客户端功能可以使用, 任何没有经过许可的客户端功能需要向服主询问。
不允许使用矿透, 仅允许世吞维护助手。
不允许投影打印机。
擅自使用未经允许的客户端将处罚手动挖掘方块(如深板岩、末地石等)。
恶意破坏服务器、卡服将直接加入黑名单, 无特殊情况永久不移除。

假人使用

玩家可以任意使用[玩家自己的id]+大于3的数字形式的假人;
[玩家自己的id]+1为其他玩家向此玩家传输属于此玩家的物品时使用, 如死亡后的尸体;
[玩家自己的id]+2为该玩家向其他玩家传输属于自己的物品时使用;
[玩家自己的id]+3为该玩家需要临时清空背包时将自己的物品寄存的假人。
除[玩家自己的id]+数字形式以外的所有假人都要在假人注册表中按格式注册。
机器挂机假人通常为[机器名]+数字形式,
若机器名称过长, 则为[主产物名]+_+数字形式
存储用假人为[存储的物品名]+数字形式, 存储假人头盔位置应戴有存储的物品一个

其他规则由服主或管理员解释。
